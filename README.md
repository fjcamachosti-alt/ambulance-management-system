# Sistema de Gestión de Ambulancias

Sistema empresarial completo para la gestión de empresas de ambulancias.

## Requisitos Previos

- Node.js 16+
- PostgreSQL 14+
- npm

## Instalación Rápida

```bash
# Backend
cd backend
npm install
npm run dev

# Frontend (en otra terminal)
cd frontend
npm install
npm start
```

## Acceso

- Frontend: http://localhost:3000
- Backend: http://localhost:5000
